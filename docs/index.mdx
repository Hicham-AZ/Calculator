# Calculator Documentation

Welcome to the Calculator project documentation! This is a Java Swing-based calculator application with advanced theming support and both standard and scientific calculation modes.

## Overview

The Calculator is a desktop application built using Java Swing that provides:

- **Standard Calculator Mode**: Basic arithmetic operations (+, -, *, /, %)
- **Scientific Calculator Mode**: Advanced functions (âˆš, power, natural logarithm)
- **Dynamic Theming**: Light and Dark themes with YAML configuration
- **Clean Architecture**: Well-structured codebase with separation of concerns
- **Comprehensive Testing**: Unit tests for calculation logic

![Calculator Screenshots](../screenshots/scientific-dark.png)

## Quick Start

```bash
# Clone the repository
git clone https://github.com/HouariZegai/Calculator.git

# Navigate to project directory
cd Calculator

# Run with Maven
mvn compile exec:java -Dexec.mainClass="com.houarizegai.calculator.App"
```
